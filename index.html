Documentation

I first started this page by adding my name as a header, add a container for the image of myself, and add a paragraph of random text to be wrapped. I then styled the image to float right in css. I then added the grid in css, creating 4 rows and 4 columns each sized correctly.

Area1

Added my picture using img tag. Set the image fit to contain in css and set the wideth and max-height to 100%.

Area2

Created two headers and an image for the phone icon. Set the display for the area to block and the headers to inline-block so they will appear side by side.

Area4

Created a table using table tag. First table row used th tag to make table header. Used table tr:nth-child(even) to set color of even rows. Used table tr:hover to set different color when hovering over entry.

Area5

Set object fit to fill for image and set width and height to 100%.

Area6

Created unordered list with ul tag. Set links to words with "a" tag.

Area7

Put a bunch of random text into a paragraph tag. Added bold, italics, and code tags to random spots in the text.

Area8

Created flexbox and set flex direction to row. Set items in flex box to be centerd on the main and cross axis. Used "a" tag and img tag to make linked images.
